# Análise de nascimentos do SINASC RONDÔNIA (Sistema de Informação sobre Nascidos Vivos) de 2019 
Projeto de análise de dados sobre nascimentos no estado de Rondônia no ano de 2019.
Para executar o programa:
\n 1 - Coloque o aquivo Início.py no mesmo local da pasta pages.
2 - Abra os arquivos:
_ Visualizar os dados
_ Mapa de nascimentos
_ Análises gráficas
Modifique o local do arquivo para onde você fez o download do .csv
3 - Abra o seu prompt de comando e digite o seguite comando:
_ cd "Caminho onde está o arquivo Início.py e a pasta Pages"
_ streamlit run Início.py

Após estes passos o web app será aberto no seu navegador padrão.
